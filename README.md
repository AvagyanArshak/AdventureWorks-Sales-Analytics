# AdventureWorks Sales Analytics

## 📊 Project Overview
SQL + Power BI Sales Dashboard using AdventureWorks2022 database.

## 🛠️ Technologies
- SQL Server 2022
- Power BI Desktop
- T-SQL (CTEs, Window Functions, Views)

## 📁 Project Structure
- `SQL/01_basic_queries.sql` — Basic SELECT, JOIN, GROUP BY queries
- `SQL/02_views.sql` — Views created for Power BI

## 📈 Dashboard Pages
1. **Executive Summary** — Total Revenue $123M, Orders, Trends
2. **Products Analysis** — Top products, Categories (Bikes 86%)
3. **Sales Team** — Linda Mitchell #1 SalesPerson
4. **Geography** — Sales by Country and Territory

## 🔍 Key Insights
- Total Revenue: $123.22M
- Total Orders: 31K
- Top SalesPerson: Linda Mitchell
- Top Category: Bikes (86%)
- Top Territory: US Northwest $18M
